Steem Notifier
==============

Requirements
------------
- Python 3.5+
- MongoDB

Run on your development machine
-------------------------------

Make sure that MongoDB is running on your computer.

- Create and activate new virtual env:

        python3 -m venv venv
        . venv/bin/activate

- Install the dependencies:
      
        pip install -r requirements.txt

- Run the app:

        ./app.py


Unit testing
------------
Run the unittest with:
    
    python -m unittest
